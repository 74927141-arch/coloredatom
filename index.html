What is Colored Atom

Native Bitcoin Colorization

Colored Atom is a fungible Bitcoin‑native asset built on the Atomicals protocol using the ARC‑20 standard. Each unit represents a real satoshi that has been deterministically marked with metadata — no wrapping, no bridges, no synthetic tokens.

1 SAT1:1 on‑chain backing
ARC‑20Atomicals standard
UTXONative ownership
L1 BTCPoW security
How it works

  1. Satoshis leave your wallet as normal Bitcoin UTXOs
  2. Atomicals attaches deterministic metadata
  3. Satoshis pass through Bitcoin consensus
  4. The same sats return — now expressed as Colored Atom

Mine COLOREDATOM

COLOREDATOM is an ARC‑20 digital token created on the Bitcoin blockchain through the Atomicals protocol. Its mining process uses computational power to deterministically release new units, anchoring issuance directly to Bitcoin Layer 1 without relying on external chains, bridges, or smart‑contract platforms.

Ecosystem

Powered by BitMeta & AVM

BitMeta is a unified platform for Bitcoin metaprotocols, bringing together ARC‑20, Ordinals, Runes, BRC‑20, Bitmap and more under a single execution environment. Rather than introducing a new chain or bridge, BitMeta extends Bitcoin itself with native programmability through the Atomicals Virtual Machine (AVM), while settling directly on Bitcoin Layer 1.

Settlement Bitcoin Layer 1
Execution AVM (Atomicals VM)
Metaprotocols ARC‑20 · Ordinals · Runes
Security Model Proof‑of‑Work
Why BitMeta Matters By unifying metaprotocol activity under a single deterministic execution engine, BitMeta removes silos between Bitcoin‑native assets. Collectors, traders, and builders can interact with tokens, digital objects, and programmable logic in one coherent system — without sacrificing Bitcoin’s security guarantees.
